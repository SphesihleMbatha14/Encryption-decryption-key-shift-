# Caesar Cipher GUI Tool  A Java Swing desktop application that provides a graphical interface for encrypting and decrypting text using the Caesar Cipher algorithm.   ## Overview This project was developed to provide a functional and clean user experience for basic cryptography. It features a modular design where encryption and decryption logic are separated into individual components, controlled by a main core frame.  ## Features * **Interactive UI:** Navigate between encryption and decryption modes seamlessly. * **Caesar Cipher Logic:** Shift text based on a user-provided numeric key. * **Dynamic Layout:** Uses Java Swing components (`JPanel`, `BorderLayout`, `GridLayout`) to manage screens without complex external dependencies. * **State Management:** Includes functionality to clear and refresh panels dynamically when switching between tools.  ## Technical Stack * **Language:** Java * **GUI:** Java Swing * **Logic:** Caesar Cipher implementation with custom character-shifting algorithms.  ## Project Structure - `theCore.java`: Main entry point and frame management. - `Display.java`: The landing screen/menu hub. - `encrypt.java`: Encryption logic and UI. - `decrypt.java`: Decryption logic and UI.  ## How to Run 1. Ensure you have the Java JDK installed. 2. Clone this repository. 3.
